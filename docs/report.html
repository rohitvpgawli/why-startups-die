Scope & methodology

Each factor is a sourced claim with an explicit confidence score. Unknown outcomes are not counted as shutdowns, and a YC batch is not treated as proof of founding year.

The active project cohort focuses on startups founded in 2021 or first institutionally funded in 2021. The reviewed archive currently contains 57 cases; the YC sampling frame is kept separate.
